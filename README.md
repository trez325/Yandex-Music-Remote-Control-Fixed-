# Yandex Music Remote Control (OPRA-Fixed)

Это форк/исправленная версия расширения **YMRC**, предназначенного для дистанционного управления Яндекс Музыкой через мобильное устройство. 

### ⚡ Что нового в этой версии:
*   **Полная адаптация под новый дизайн:** Исправлены селекторы кнопок, которые перестали работать после обновления интерфейса Яндекс Музыки (2024-2025).
*   **Управление громкостью:** Добавлена поддержка команд изменения громкости (в дополнение к стандартным Play/Pause и переключению треков).
*   **Оптимизация:** Исправлены ошибки в `CommandExecutor`, из-за которых команды могли дублироваться или не доходить.

---

## Как установить

Поскольку это кастомная версия, её нужно устанавливать в режиме разработчика:

1.  **Скачайте репозиторий:** Нажмите зеленую кнопку `Code` -> `Download ZIP` (или склонируйте через git).
2.  **Распакуйте архив** в удобную папку.
3.  **Откройте браузер** (Chrome, Edge, Opera или Яндекс.Браузер).
4.  Перейдите по адресу: `chrome://extensions/`
5.  Включите **"Режим разработчика"** (Developer mode) в правом верхнем углу.
6.  Нажмите кнопку **"Загрузить распакованное расширение"** (Load unpacked) и выберите папку, в которую вы распаковали файлы.

---

## Как пользоваться

1.  Откройте [Яндекс Музыку](https://music.yandex.ru) в браузере.
2.  Нажмите на иконку расширения в панели инструментов.
3.  Отсканируйте появившийся **QR-код** телефоном или перейдите по ссылке под ним.
4.  На открывшейся странице пульта вы сможете управлять воспроизведением.
